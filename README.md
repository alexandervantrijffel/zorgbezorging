## Getting Started

#### 1. Clone and Install

```
# install dependencies
yarn install
```

```bash
# Install Pods
cd ios && pod install
```

```
# start the metro server
yarn start
```

#### 2. Open RNS in your iOS simulator

Run this command to start the development server and to start your app on iOS simulator:
```
yarn run:ios
```

Or, if you prefer Android:
```
yarn run:android
```
